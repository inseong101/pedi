Codex 대시보드 사용 가이드

저장소를 GitHub Pages에 올렸다면 아래 단계에 따라 Codex 대시보드를 바로 활용할 수 있습니다. 각 단계는 실제 페이지 구성과 연결되어 있어, 그대로 따라 하면 누구든지 빠르게 정리된 자료를 확인할 수 있습니다.

  1. 배포 확인

    Settings → Pages에서 Deploy from a branch를 선택하고 main과 /root를 지정합니다. 주소는 https://<깃허브아이디>.github.io/pedi/와 같이 생성되며, 몇 분 후 접속하면 최신 목차가 자동으로 로드됩니다.

  2. 데이터 갱신

    chapter/ 디렉터리에는 장별 Markdown이, question_bank.json에는 문제 메타데이터가 들어 있습니다. 파일을 수정하거나 새 문제를 추가한 뒤 커밋하면, 다음 배포에서 Codex 검색 인덱스가 자동으로 재생성됩니다.

  3. 검색과 딥링크

    상단의 “개념/문제 검색” 상자에 키워드를 입력하면 장·절·항목·문제가 묶인 카드가 표시됩니다. 카드의 “바로 가기” 링크를 누르면 해당 목차 섹션으로 스크롤되며, 연도 칩을 클릭하면 연도별 출제 내역만 모아볼 수 있습니다.

Tip. 로컬에서도 확인하고 싶다면 python -m http.server로 간단한 서버를 띄우고 http://localhost:8000에 접속하세요. GitHub Pages와 동일한 구성이 로딩됩니다.

검색어를 입력하면 관련 장·절·항목과 문제 수가 정리되어 표시됩니다.
